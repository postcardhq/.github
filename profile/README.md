# Postcard

> Democratizing the truth.

Postcard is a digital forensics tool that traces social media posts back to their definitive origin—calculating a credibility score by auditing how much content has drifted from the ground truth.

## What We Do

- **OSINT Pipeline**: 4-stage forensic analysis that traces any social media post to its source
- **Postcard Score**: 0-100% credibility rating based on content drift from truth
- **AI-Powered Verification**: Gemini-powered corroboration and verification agents

## Tech Stack

- Next.js 16 | TypeScript | AI SDK 6.0
- Gemini 2.0 | Drizzle ORM | Turso (SQLite)
- Playwright | OpenAI | oEmbed APIs
